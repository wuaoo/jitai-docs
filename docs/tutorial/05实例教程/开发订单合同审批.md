# 开发订单合同审批

该教程通过讲解开发订单合同审批的过程，带你逐步理解审批流程、文件模板、打印的基本操作。

# 期望效果

1.  审批流程

2.  一键生成合同

3.  打印合同


# 思路分析

1.  将线下的审批流程，每个环节配置一个节点，按权限配置。

2.  审批人，尽量使用 动态变量，例如 ： 角色、 发起人/成员字段/部门字段的 N 级主管。
